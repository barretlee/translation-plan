# 翻译计划

今年计划翻译 100 篇左右的好文，前期主要是英译中，后期会是中译英。

## 策略

每天从 RSS 订阅中找出几篇不错的文章贴在 issue，

- 感觉内容超过 80 分的文章，会翻译出来，预发布到 issue，最后沉淀在[这里](https://www.barretlee.com/translation/)；
- 感觉不足 80 分的文章，简单评论给出概要后将 issue 关闭。
